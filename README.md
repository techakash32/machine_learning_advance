# 🚀 Machine Learning Advance

This repository contains my practical work on **Machine Learning model deployment**, **Flask API development**, and **API testing**.
The main goal of this project is to understand how a trained ML model is converted into a real-world usable API.

---

## 📌 Project Overview

In this project, I built a complete machine learning workflow starting from model training to API deployment using Flask. The project demonstrates how machine learning models can be integrated into backend applications and accessed through REST APIs.

The system allows users to send input data through an API request and receive predictions generated by a trained machine learning model.

---

## 🧠 Key Concepts Covered

* Machine Learning Model Training
* Data Preprocessing & Encoding
* Handling Missing Values
* SMOTE for Imbalanced Data
* Model Serialization using Joblib
* Flask API Development
* REST API Design
* API Testing (Postman / JSON Requests)
* Model Prediction via HTTP Requests

## 🔄 Workflows

1. Load dataset
2. Perform preprocessing and encoding
3. Handle missing values
4. Apply SMOTE for class balancing
5. Train Random Forest model
6. Save model using Joblib
7. Create Flask API
8. Send prediction requests using JSON
9. Return prediction response

## ▶️ How to Run the Project

### 1️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 2️⃣ Train Model

```
python main.py
```

### 3️⃣ Run Flask API

```
python app.py
```

Server will start at:

```
http://127.0.0.1:5000
```

## 📡 API Endpoint

### POST `/predict`

#### Request Body (JSON)

```
{
  "age": 45,
  "gender": "Male",
  "fever": 101,
  "cough": "Mild",
  "city": "Delhi"
}
```

#### Response

```
{
  "prediction": 1
}
```
